Configuration
*****************
The file build.TEMPLATE_USERID.properties is used for "personal" build.
If your log in id is mike then  this file should be named
build.mike.properties to host your personal settings.

You have to edit this file according to your personal settings set up.

Build layout templates:
***********************
1. Go to folder ...\gothia-forum-theme\trunk\layouttpl\gothia-layouts-layouttpl
2. run the command: ant war

Build layout themes:
***********************
1. Go to folder C:\projects\gothia-forum-theme\trunk\themes\gothia-forum-theme
2. run the command: ant war
